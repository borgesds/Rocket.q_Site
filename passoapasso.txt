Todos os css e html são recortados e colocados dentro de uma nova pasta chamada htmls


npm init -y

===> install modulos
npm install ejs

npm install express

===> cria uma pasta
criar pasta 'src'
'src e onde vai ficar todo o codigo agora'
    >criar um arquivo JS dentro da pasta chamado server.js(o node vai rodar dentro de um servidor)

===> Arquivo server <===
dentro do server.js

    const express = require('express');

    const server = express();

    server.listen(3000, () => console.log("RODANDO"))

para rodar o server depois de configurar
    >node src/server.js

===> Alteração no package.json <===
deleta o test
    >"test": "echo \"Error: no test specified\" && exit 1"

coloque no lugar starr, ele vai da boot mais rapido em alguns comandos
    >"start": "node src/server.js"

    OBS: agora no terminal você vai escrever "npm start", ele subistiui o comando que vc atribuiu.

===> Criando uma nova pasta <===
dentro da pasta src crie uma pasta chamada 'view' onde vai conter os arquivos novos de html
    >crie um arquivo 'index.html'
        >como o node não roda direto no browser troque a extensão do arquivo para ejs


            >OBS:os arquivos devem rodar no browser http://localhost:3000
